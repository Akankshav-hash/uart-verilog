# UART Transmitter and Receiver using Verilog

## Overview

This project implements a Universal Asynchronous Receiver Transmitter (UART) using Verilog HDL. The design includes both transmitter and receiver modules and demonstrates asynchronous serial communication through a loopback configuration.

## Features

* UART Transmitter (TX)
* UART Receiver (RX)
* 8-bit data transmission
* Start and stop bit generation
* Loopback verification
* Finite State Machine (FSM) based implementation

## Project Structure

* `src/` – Verilog source code
* `tb/` – Testbench
* `docs/` – Waveforms and block diagrams
* `report/` – Assignment report

## Tools Used

* Verilog HDL
* Cadence EDA Tools

## How to Run

1. Compile the UART module and testbench.
2. Execute the simulation.
3. Verify transmitted and received data through waveforms.

## Results

Simulation confirms successful serial transmission and reception of data using the internal loopback mechanism.

## Applications

* Embedded systems
* Microcontroller communication
* Serial debugging
